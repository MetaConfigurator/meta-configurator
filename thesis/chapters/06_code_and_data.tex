\chapter{Code and Data Availability}\label{sec:code_and_data}

\section{Data Availability}

The JSON and JSON-LD data examined in Chapters~\ref{sec:background} and~\ref{sec:design} are publicly available on GitHub in the repository \texttt{NFDI4IngModelValidationPlatform}\footnote{\url{https://github.com/BAMresearch/NFDI4IngModelValidationPlatform}}.
The repository contains workflows and documentation for the validation of constitutive models.

JSON data presented in the \gls{mof} synthesis study in Chapter~\ref{sec:application} are also publicly available on GitHub in the repository \texttt{mof-synthesis-data-modeling}\footnote{\url{https://github.com/FAIRChemistry/mof-synthesis-data-modeling}}, with some modifications to comply with the intended \gls{rml} mapping.

Both repositories are distributed under the MIT License.

\section{Code Availability}

The source code of MetaConfigurator project is publicly available on GitHub in the repository \texttt{meta-configurator}\footnote{\url{https://github.com/MetaConfigurator/meta-configurator}}.
