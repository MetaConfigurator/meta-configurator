\section*{Abstract}

Scientific workflows increasingly produce structured JavaScript Object Notation (JSON) data. Although this data is easy to exchange, it is still hard to interpret consistently across systems at the semantic level. JSON Schema helps with structural validation, but it does not provide native Linked Data semantics. This thesis addresses that gap by extending MetaConfigurator, a schema-driven web application for structured data authoring, with Semantic Web capabilities.

The main contribution is an integrated Resource Description Framework (RDF) Authoring Panel that supports end-to-end semantic data workflows in a single user interface. Researchers can start from existing hierarchical or tabular research data, use AI-assisted RDF Mapping Language (RML) to transform it into RDF, and then continue in the same environment to refine triples, run SPARQL queries, visualize knowledge graphs, and export to common RDF serializations.

Overall, the result is a unified workflow that connects traditional non-linked research data with Semantic Web infrastructures. The thesis demonstrates this workflow on a real Metal-Organic Framework (MOF) synthesis application, showing how tabular data can be transformed into RDF, iteratively improved, explored, visualized, and integrated into a target knowledge store.

\section*{Kurzfassung}

Wissenschaftliche Workflows erzeugen zunehmend strukturierte Daten im JavaScript Object Notation (JSON)-Format. Obwohl diese Daten leicht austauschbar sind, lassen sie sich auf semantischer Ebene über verschiedene Systeme hinweg nur schwer konsistent interpretieren. JSON Schema unterstützt die strukturelle Validierung, bietet jedoch keine nativen Linked-Data-Semantiken. Diese Arbeit schließt diese Lücke, indem sie MetaConfigurator, eine schema-gesteuerte Webanwendung zur Erstellung strukturierter Daten, um Funktionen des Semantic Web erweitert.

Der Hauptbeitrag ist ein integriertes Authoring-Panel für das Resource Description Framework (RDF), das durchgängige semantische Daten-Workflows innerhalb einer einzigen Benutzeroberfläche ermöglicht. Forschende können von bestehenden hierarchischen oder tabellarischen Forschungsdaten ausgehen, mithilfe KI-gestützter RDF Mapping Language (RML) diese in RDF transformieren und anschließend in derselben Umgebung Tripel verfeinern, SPARQL-Abfragen ausführen, Wissensgraphen visualisieren und in gängige RDF-Serialisierungen exportieren.

Insgesamt entsteht so ein einheitlicher Workflow, der traditionelle, nicht verknüpfte Forschungsdaten mit Semantic-Web-Infrastrukturen verbindet. Die Arbeit demonstriert diesen Workflow anhand einer realen Anwendung zur Synthese von Metal-Organic Frameworks (MOFs) und zeigt, wie tabellarische Daten in RDF überführt, iterativ verbessert, untersucht, visualisiert und in einen Ziel-Wissensspeicher integriert werden können.
